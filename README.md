# RareBus

**You are a transit sicko. Collect the routes that prove it.**

RareBus is a tiny, mean-spirited, strangely compelling browser game about logging the most unhinged, obscure, and cursed public transit rides on Earth. Some routes are common. Some are legendary. A few should probably be classified as crimes against humanity.

- Hand-written database of routes across multiple cities, each with real personality
- Rarity system from Common to Cursed
- Daily Ride — one special, higher-rarity route per calendar day
- Streak tracking and a Sicko Score (completely made-up, deeply meaningful)
- Flag the routes you've actually ridden, for the truly unwell
- Share your collection as a beautiful brag

This is the fun, stupid sibling to serious transit tools. No real data. No noble goals. Just the quiet joy of becoming the kind of person who has strong feelings about the 3:17am 301.
